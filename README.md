# Sudoku Game

A cross-platform Sudoku game for iOS and Android.

## Overview

This is a mobile-friendly Sudoku game built with HTML, CSS, and JavaScript, designed to work seamlessly on both iOS and Android devices.

## Features

- Classic Sudoku gameplay
- Mobile-optimized interface
- Touch-friendly controls
- Responsive design for various screen sizes

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Progressive Web App (PWA) capabilities for mobile deployment

## Getting Started

1. Clone this repository
2. Open `index.html` in a web browser or deploy to a web server
3. For mobile deployment, use tools like Cordova or Capacitor to package as native apps

## Development

The project structure:
- `index.html` - Main game interface
- `style.css` - Game styling
- `script.js` - Game logic
- `manifest.json` - PWA configuration
